    Nico's Ritual (Global Game Jam Barcelona 2016)

    Game made at the Global Game Jam Barcelona 2016.

    You must serve the costumers as they arrive. Each of them will ask for a drink, which needs to be made using a specific ritual (recipe). You should guess the ingredients by the looks of the costumer!

    Tools and technologies:

    Unity, Blender, Gimp, Photoshop

    My participation:

    I made the UI, the sounds and some assets of the bar like the jukebox, the glasses and the bottles.

    Things learned:

    It was my first "game". I've learned team work and a lot of videogame development things like normals, canvas and triggers.

    Links:
    Global Game Jam game site.

    PuzzleBubble (UPC computer grade Videogames subject)

    Game made at the UPC computer grade Videogames subject.

    Version of the classic game Puzzle Bubble. The goal is join 3 or more balls with the same color until the stage is clear. The scene will become smaller as the time passes. The black ball is unstoppable, use it wisely!

    Tools and technologies:

    C++, OpenGL

    My participation:

    Coding, debugging

    Things learned:

    Tile mapping, texturing in OpenGL, good memory management, debugging, memory leak analysis and fixing them.

    Game binaries (Windows).
